# What is Windows?

Microsoft Windows is one of the most widely used **operating systems** in the world.  
It provides a **graphical user interface (GUI)** that allows users to interact with the computer without using command-line instructions.

Windows manages:

- hardware resources
- files and folders
- user accounts
- applications and processes

---

# Windows Versions

Some common Windows versions include:

- Windows XP
- Windows 7
- Windows 8 / 8.1
- Windows 10
- Windows 11

In professional environments, Windows systems are often integrated into **Active Directory domains**.

---

# The Windows Desktop

The desktop is the main interface users interact with.

Main components:

### Start Menu
The central location to:

- launch applications
- access settings
- shut down or restart the computer
- search for files and programs

### Taskbar
Displays:

- running applications
- pinned programs
- system tray icons
- notifications
- clock

### System Tray
Located on the right side of the taskbar.

Contains icons for:

- network
- volume
- antivirus
- background applications

---

# File System

Windows uses the **NTFS (New Technology File System)** by default.

Key features:

- file permissions
- encryption
- compression
- journaling for reliability

Files are organized using **directories (folders)**
