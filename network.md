## Overview
The OSI (Open Systems Interconnection) model is a conceptual framework used to understand network communication.

It is divided into 7 layers:

1. Physical
2. Data Link
3. Network
4. Transport
5. Session
6. Presentation
7. Application

---

## Layer Breakdown

### 1. Physical Layer
- Transmits raw bits over a physical medium
- Cables, voltage levels, signals

### 2. Data Link Layer
- MAC addresses
- Frames
- Error detection

### 3. Network Layer
- IP addressing
- Routing
- Packets

### 4. Transport Layer
- TCP / UDP
- Ports
- Reliability

### 5. Session Layer
- Manages sessions between systems

### 6. Presentation Layer
- Encryption / Decryption
- Data formatting

### 7. Application Layer
- HTTP
- FTP
- SMTP
