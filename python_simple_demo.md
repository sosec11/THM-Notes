## Introduction

Python is a popular programming language widely used in **cybersecurity, automation, scripting, and data analysis**.

This module introduces basic Python concepts and demonstrates how simple scripts can be written and executed.

Python is especially useful in cybersecurity for:

- automation
- scripting tasks
- data processing
- building security tools

---

## What is Python?

Python is a **high-level, interpreted programming language** known for its simplicity and readability.

It allows developers to write clean and concise code.

Example of a simple Python script:

```python
print("Hello, World!")
```

This command displays text in the terminal.

---

## Running Python Code

Python scripts can be executed in different ways:

- using the Python interpreter
- running a `.py` file

Example:

```bash
python script.py
```

This executes the Python script named `script.py`.

---

## Variables

Variables are used to store data.

Example:

```python
name = "Alice"
age = 25
```

Variables can store different types of data such as:

- strings
- integers
- booleans

---

## Basic Data Types

Common Python data types include:

- **string** → text (`"Hello"`)
- **integer** → numbers (`10`)
- **float** → decimal numbers (`3.14`)
- **boolean** → true/false (`True`, `False`)

Example:

```python
message = "Hello"
number = 10
pi = 3.14
is_active = True
```

---

## User Input

Python allows users to provide input.

Example:

```python
name = input("Enter your name: ")
print("Hello " + name)
```

This script asks the user for input and displays a message.

---

## Basic Operations

Python supports basic operations such as:

```python
a = 5
b = 3

print(a + b)  # addition
print(a - b)  # subtraction
print(a * b)  # multiplication
print(a / b)  # division
```

---

## Conditional Statements

Conditional statements allow the program to make decisions.

Example:

```python
age = 18

if age >= 18:
    print("Adult")
else:
    print("Minor")
```

---

## Loops

Loops allow repeating actions.

### For loop

```python
for i in range(5):
    print(i)
```

### While loop

```python
count = 0

while count < 5:
    print(count)
    count += 1
```
