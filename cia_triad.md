# TryHackMe - CIA Triad

## Introduction

The **CIA Triad** is a fundamental model in cybersecurity that represents three core principles used to protect information systems:

- **Confidentiality**
- **Integrity**
- **Availability**

These three pillars guide how security systems are designed and implemented.

---

## Confidentiality

Confidentiality ensures that information is only accessible to authorized individuals.

It protects sensitive data from unauthorized access or disclosure.

Examples:

- encryption (e.g., HTTPS)
- authentication systems (passwords, MFA)
- access controls

If confidentiality is compromised, sensitive information can be exposed.

---

## Integrity

Integrity ensures that data remains accurate and unchanged unless modified by authorized users.

It protects data from unauthorized modification or corruption.

Examples:

- hashing
- checksums
- digital signatures

If integrity is compromised, data can be altered or manipulated.

---

## Availability

Availability ensures that systems and data are accessible when needed.

It focuses on maintaining reliable access to resources.

Examples:

- backups
- redundancy
- protection against DDoS attacks

If availability is compromised, systems may become unusable.

---

## Why the CIA Triad Matters

The CIA Triad is used to:

- design secure systems
- evaluate security risks
- implement security controls

Security professionals must balance all three aspects, as improving one may affect the others.

---

## Examples in Real Life

- Confidentiality: encrypting user passwords
- Integrity: verifying file integrity with hashes
- Availability: ensuring a website stays online during high traffic

---

## Key Takeaways

- The CIA Triad consists of Confidentiality, Integrity, and Availability.
- These principles form the foundation of cybersecurity.
- Each pillar addresses a different aspect of system protection.
- Effective security requires balancing all three.

---

## Conclusion

The **CIA Triad** is a core concept in cybersecurity that helps define how systems should be protected.

Understanding these principles is essential for identifying risks, implementing protections, and designing secure infrastructures.
